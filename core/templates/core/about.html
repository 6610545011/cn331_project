About the Project

sTUdyPlan is a web application designed to help Thammasat University students plan their studies efficiently. Users can search for reviews, write reviews about courses and professors, and help their peers make informed decisions when selecting courses and instructors. This project was developed as part of the Software Engineering (CN331) course.
