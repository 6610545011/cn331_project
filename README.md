# sTUdyPlan - แพลตฟอร์มวางแผนและรีวิวรายวิชา มหาวิทยาลัยธรรมศาสตร์

**sTUdyPlan** คือเว็บแอปพลิเคชันที่พัฒนาขึ้นด้วย Django เพื่อช่วยให้นักศึกษามหาวิทยาลัยธรรมศาสตร์สามารถวางแผนการเรียนและตัดสินใจเลือกลงทะเบียนรายวิชาได้อย่างมีประสิทธิภาพมากขึ้น ผ่านระบบการรีวิวและวางแผนตารางเรียนที่ขับเคลื่อนโดยชุมชนนักศึกษา

โปรเจกต์นี้เป็นส่วนหนึ่งของรายวิชา วพ.331 วิศวกรรมซอฟต์แวร์ (Software Engineering)

## เป้าหมายของโปรเจกต์

จากปัญหาที่นักศึกษาไม่สามารถทราบถึงรายละเอียดเชิงลึกของแต่ละรายวิชา เช่น สไตล์การสอนของอาจารย์, แนวทางการให้คะแนน, หรือบรรยากาศในห้องเรียน ทำให้การวางแผนการเรียนเป็นไปได้ยาก sTUdyPlan จึงถูกสร้างขึ้นมาเพื่อแก้ปัญหาเหล่านี้ โดยมีเป้าหมายหลักคือ:

1.  **สร้างศูนย์กลางข้อมูลรีวิว:** เป็นพื้นที่ให้นักศึกษาได้แบ่งปันประสบการณ์และแสดงความคิดเห็นต่อรายวิชาและอาจารย์ผู้สอน
2.  **ช่วยในการตัดสินใจ:** ทำให้นักศึกษาสามารถค้นหาข้อมูลและเปรียบเทียบรายวิชาที่ตรงกับสไตล์การเรียนรู้ของตนเองได้
3.  **อำนวยความสะดวกในการวางแผน:** มีเครื่องมือช่วยวางแผนตารางเรียน (Schedule Planner) เพื่อให้นักศึกษาสามารถจัดตารางเรียนล่วงหน้าได้อย่างเป็นระบบ


## Key Features

-   **ระบบยืนยันตัวตน (Authentication):**
    -   เพื่อยืนยันว่าผู้รีวิวมีตัวตน และลงเรียนในรายวิชานั้น ๆ จริง ๆ
-   **หน้าเว็บหลักและข้อมูลโปรเจกต์:**
    -   Homepage แสดงข้อมูลเบื้องต้น
    -   หน้า About ที่แนะนำทีมผู้พัฒนาและรายละเอียดโปรเจกต์
-   **ระบบรีวิวรายวิชา:**
    -   ค้นหารายรีวิวรายวิชา
    -   อ่านรีวิวจากนักศึกษาคนอื่นๆ
    -   เขียนรีวิวและให้คะแนนรายวิชาและอาจารย์ผู้สอน
-   **ระบบวางแผนตารางเรียน:**
    -   เครื่องมือสำหรับจัดตารางเรียน
-   **ระบบบุ๊กมาร์ค:**
    -   สามารถบุ๊กมาร์ครายวิชา หรือรีวิวที่น่าสนใจ


## โครงสร้างโปรเจกต์ (Project Structure)

โปรเจกต์นี้ถูกแบ่งออกเป็นแอปพลิเคชันย่อยๆ ตามหลักการของ Django เพื่อให้ง่ายต่อการจัดการและพัฒนา:

-   **'core'**: จัดการหน้าเว็บทั่วไปที่ไม่มีความซับซ้อนและการแสดงผลอื่น ๆ ได้แก่ Homepage, Contact, Search
-   **'users'**: จัดการทุกอย่างที่เกี่ยวกับผู้ใช้ คือ ระบบ Login/Logout และโปรไฟล์ รวมไปถึงการ Bookmark
-   **'review'**: เป็นแอปหลักสำหรับจัดการการแสดงความคิดเห็นของผู้ใช้งานต่อแต่ละรายวิชา
-   **'planner'**: จัดการระบบวางแผนตารางเรียนของผู้ใช้งาน

## สำหรับการติดตั้งแบบ local

1. เข้าไปที่ Folder ที่ต้องการจะเก็บแอปพลิเคชันและพิมพ์คำสั่งนี้ลงใน terminal

    ```
    git clone https://github.com/6610545011/cn331_project.git
    ```
1. สร้างและ activate virtual environment ของคุณ *optional*
1. พิมพ์คำสั่งนี้ลงใน terminal เพื่อสร้างฐานข้อมูล บัญชีแอดมิน และรัน server

    ```
    pip install -r requirements.txt
    python manage.py migrate
    python manage.py createsuperuser
    // create your admin username, email and password

    python manage.py runserver
    ```
1. เปิด http://127.0.0.1:8000/ เพื่อใช้งาน

**นี่เป็นการใช้งานแบบ local ไม่สามารถส่ง url http://127.0.0.1:8000/ ให้ผู้อื่นใช้งานด้วยได้**
